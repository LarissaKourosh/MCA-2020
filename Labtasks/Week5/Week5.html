Week 5 - Standards in Curation

Task 1 - Schema of Metadata Elements - Describing Music

As demonstrated below, I have created a metadata scheme that I believe contains all the key components in describing music within a 1000-song data set. The data within the scheme not only contains data related to that of the original file, such as artist, album/song titles etc. but also data related to the transcription of the file/song. Including all these types of information means I am providing information not only on the original file but also the encoding and editing information in relation to the project

title transcription song title
album title original album title
artist original song artist
release date original song release date
composer person who created original song
publisher names of entites responsible for publication of the file
publisher location geographic location of the publisher
date of publication date of publication of the file
source of publication source of the original file
size of file size of the file
copyright/use restrictions copyright restrictions
producer those involved in production of file
encoding encoding details - changes made
edition edition of the file (version of file)
